# API POINTS




## ARTIKEL
### Menampilkan seluruh artikel
```
GET: /artikel
response:
[
  {
    "id"        : "",
    "judul"     : "",
    "desc"      : "",
    "tanggal"   : "",
    "img"       : "",
    "konten"    : ""
  },
  {
    ...
  }
]
```
### Menampilkan artikel berdasarkan ```id```
```
GET: /artikel/[id]
response:
{
    "id"        : "",
    "judul"     : "",
    "desc"      : "",
    "tanggal"   : "",
    "img"       : "",
    "konten"    : ""
}
```
### Menambahkan artikel
```
POST: /artikel
data:
{

    "judul"     : "",
    "desc"      : "",
    "tanggal"   : "",
    "img"       : "",
    "konten"    : ""
}
response:
true    //if true
false   //if false
```
### Mengubah artikel
```
PUT: /artikel
data:
{

    "judul"     : "",
    "desc"      : "",
    "tanggal"   : "",
    "img"       : "",
    "konten"    : ""
}
response:
true    //if true
false   //if false
```
### Menghapus artikel
```
DELETE: /artikel
response:
true    //if true
false   //if false
```
